# claude-skill-claude-code-setup

A [Claude Code](https://claude.ai/code) skill that guides the complete first-time setup of Claude Code on a new machine.

## What it does

Walks through every step of getting Claude Code ready to use productively:

1. **Installation** — macOS, Linux, ChromeOS, Windows
2. **API key** — interactive or environment variable
3. **First run** — key commands and shortcuts
4. **CLAUDE.md** — project context file (auto-generated via `/init`)
5. **Settings** — `settings.json`, permission modes, model selection
6. **Memory system** — persistent cross-session memory
7. **Skills & plugins** — how to install and which to start with
8. **IDE integration** — VS Code and JetBrains extensions

## Install

```bash
claude skill install https://github.com/giandemoncell-prog/claude-skill-claude-code-setup
```

Then in any Claude Code session:

> "set up Claude Code on this new machine"  
> "help me configure Claude Code for the first time"  
> "how do I get started with Claude Code?"

## Who this is for

- First-time Claude Code users who want a guided setup
- Developers onboarding a new machine to their existing Claude Code workflow
- Teams standardizing their Claude Code configuration
